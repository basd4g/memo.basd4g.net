# memo.basd4g.net

[my web site](https://memo.basd4g.net) with [Hugo](https://gohugo.io/)

## Setup

```sh
$ brew install hugo # macOS
# or
$ sudo apt install -y hugo # Ubuntu

$ git clone --recursive https://github.com/basd4g/memo.basd4g.net
$ cd memo.basd4g.net

# Running locally
$ hugo server -D

# Create new post
$ hugo new posts/hoge.md

```
